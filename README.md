# Job Application Tracker (Java Swing)

A basic desktop application built using Java Swing.  
Users can enter job application details, which are saved to a file and can be reviewed later.

## Features
- User input form with:
  - Name, gender, job type, location
  - Skills and comments
- File writing (`Application.txt`) to store applications
- View previous records using a separate viewer window

## Technologies
- Java
- Swing (GUI)
- File I/O

## Note
This was created as part of a university lab task to practice OOP concepts and GUI programming in Java.
